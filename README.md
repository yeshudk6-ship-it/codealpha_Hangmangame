# 🎮 CodeAlpha - Hangman Game

## 📌 Project Overview

This project is developed as part of the **CodeAlpha Python Programming Internship**.

The Hangman Game is a simple word-guessing game built using Python. The player has to guess a hidden word one letter at a time before running out of attempts. The game demonstrates the use of Python fundamentals, including loops, conditional statements, lists, dictionaries, functions, and the `random` module.

---

## 🚀 Features

* Random word selection
* Hint for every word
* 6 incorrect guess limit
* Input validation
* Prevents duplicate guesses
* Displays guessed letters
* ASCII Hangman drawing
* Win/Lose messages
* Scoreboard
* Play Again option
* Beginner-friendly code

---

## 🎮 How to Play

1. Start the game.
2. Read the hint provided.
3. Enter one letter at a time.
4. Correct letters will appear in the hidden word.
5. Incorrect guesses reduce your remaining attempts.
6. Guess the complete word before all attempts are used.
7. Choose **Play Again** to start a new game.
